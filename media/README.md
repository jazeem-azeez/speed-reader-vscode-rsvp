# Media Assets

## Icon

The extension requires a 128x128 PNG icon for the marketplace.

Current file: `icon.svg` - A simple SVG icon that should be converted to PNG.

To create the PNG icon:
1. Open `icon.svg` in an image editor
2. Export as PNG at 128x128 pixels
3. Save as `icon.png` in this directory

The icon should:
- Be 128x128 pixels
- Have a transparent or solid background
- Be recognizable at small sizes
- Follow VS Code marketplace guidelines
