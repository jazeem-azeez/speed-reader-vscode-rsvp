# Change Log

All notable changes to the RSVP Speed Reader extension will be documented in this file.

## [0.1.0] - 2026-01-17

### Added
- Initial release of RSVP Speed Reader
- Support for Markdown, plain text, PDF, and EPUB files
- ORP (Optimal Recognition Point) highlighting at 38% of word length
- Keyboard-first controls (space to pause, arrows for speed, etc.)
- Progress bar and reading statistics
- Context menu integration for file explorer
- Configurable WPM, chunk size, and font family
- Performance-optimized text processing with streaming support
- Memory-efficient chunking for large files
